# Full-Stack Contact Manager

A full-stack web application that allows users to securely manage personal contacts through authentication, database-backed storage, and dynamic CRUD operations.

## Overview

This application provides a simple but functional system for managing contacts, including account creation, login, and full CRUD (Create, Read, Update, Delete) functionality. It is designed to demonstrate backend development, database integration, and user authentication in a real-world web application setting.

## Features

- User authentication (register & login)
- Secure session handling
- Create, view, update, and delete contacts
- Persistent data storage using MySQL
- Dynamic front-end updates using JavaScript
- Clean and responsive UI

## Architecture

The application follows a standard client-server architecture:

- The frontend handles user interaction and sends requests
- The PHP backend processes requests and handles business logic
- MySQL stores user accounts and contact data
- API endpoints connect the frontend and backend

## Key Functionality

- Authentication system with user session management
- CRUD operations connected to a relational database
- Server-side validation and request handling
- Database schema design for structured contact storage
